# SkyCast 🌤️

SkyCast is a modern, responsive weather application built with Vanilla JavaScript, HTML, and CSS. It provides real-time weather data and a 7-day forecast for any city worldwide.

![SkyCast UI](https://raw.githubusercontent.com/rahmanmdmustafizur958-ai/Weather-API/main/index.html) <!-- Note: This is a placeholder, as there are no actual images in the repo yet -->

## ✨ Features

- **🔍 Global Search:** Instantly find weather information for any city using OpenStreetMap's geocoding.
- **🌡️ Real-time Data:** Get current temperature, "feels like" temperature, humidity, wind speed, and visibility.
- **📅 7-Day Forecast:** Plan ahead with a detailed daily forecast including high and low temperatures.
- **🎨 Dynamic Themes:** The app's interface automatically adjusts its color palette based on the current temperature (Warm, Neutral, or Cold).
- **📱 Responsive Design:** Fully optimized for all devices, from desktop monitors to mobile phones.
- **💎 Premium UI:** Built with a glassmorphism design aesthetic, smooth micro-animations, and Lucide icons.

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Icons:** [Lucide Icons](https://lucide.dev/)
- **Typography:** [Outfit (Google Fonts)](https://fonts.google.com/specimen/Outfit)
- **APIs:**
  - [Open-Meteo API](https://open-meteo.com/) for weather data.
  - [OpenStreetMap Nominatim](https://nominatim.org/) for geocoding.
